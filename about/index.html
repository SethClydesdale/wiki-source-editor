About Wiki Source Editor

What is Wiki Source Editor?

Wiki Source Editor is an Open Source tool that allows you to view and edit the source code of any public wiki on Github. More importantly, it gives you the ability to share your modifications with others, so that you can collaborate on a wiki together.

Why was Wiki Source Editor Made?

I created Wiki Source Editor, because I needed a way to collaborate on Github wikis with clients who didn't use Github or have a Github account. It allows the user to share their changes via a URL, which gives you the power to propose changes to a wiki without being a collaborator or owning a Github account.
